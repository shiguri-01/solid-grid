# Installation

## Requirements

- [SolidJS](https://www.solidjs.com/) >= 1.9.0

## Install

```bash
npm install @shiguri/solid-grid
# or
pnpm add @shiguri/solid-grid
# or
bun add @shiguri/solid-grid
```

## TypeScript

Type definitions are included. No additional `@types` packages needed.
